# Zakaz1 — Supabase rebuild

Готовая версия под существующий проект Supabase.

## Что нужно сделать
1. Залить все файлы в корень репозитория.
2. В Vercel добавить переменные окружения из `.env.local.example`.
3. Задеплоить проект.

## Админка
- URL: `/admin/login`
- PIN берётся из `NEXT_PUBLIC_ADMIN_PIN`

## Что использует проект
- branches
- menu_items
- orders
- order_items
- stop_list

Проект пытается адаптироваться к имеющейся схеме таблиц по первым строкам данных и типовым названиям колонок.
