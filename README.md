# На Виражах — пересобранный проект

Что внутри:
- клиентская страница заказа
- админка `/admin`
- стоп-лист
- корзина
- заказы и статусы
- локальная синхронизация между вкладками через `localStorage` + `BroadcastChannel`
- звук готовности заказа через Web Audio API без обязательного mp3

## Как заливать
1. Очистить старый репозиторий или заменить файлы целиком.
2. Загрузить содержимое архива в корень проекта.
3. Запустить деплой.

## Структура
- `app/` — страницы Next.js
- `components/` — UI-компоненты
- `hooks/` — клиентские хуки
- `lib/` — данные и хранилище

## Важно
Это локальная версия без Supabase. Для разных устройств через интернет потом можно сделать отдельную версию с серверной синхронизацией.
